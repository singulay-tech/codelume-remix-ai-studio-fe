# 码镜工坊帮助文档

## 简介

码镜工坊是 **码镜（Codelume）** 的壁纸编辑器软件，用于创作符合码镜协议的动态动态壁纸，最终导出为 `.bundle` 格式, 供码镜壁纸软件导入使用。

## 快速开始
