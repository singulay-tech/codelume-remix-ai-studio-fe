# 码镜工坊（Codelume Studio）

## 产品定位

码镜工坊是面向创作者的壁纸内容生产工具，用于场景编辑、资源编排与壁纸包导出。目标是降低动态壁纸制作门槛，让非程序用户也能完成可交付的作品。

## 核心能力

- 场景编辑：通过可视化方式调整节点、图层、位置与时间线。
- 资源管理：统一管理纹理、音频、视频、字体和 shader 资源。
- 实时预览：编辑后立即查看效果，减少反复导出验证成本。
- 壁纸包导出：按协议导出 `.bundle`，可直接被码镜客户端加载。

## 导出流程

1. 完成场景配置与资源检查。
2. 补充元信息（名称、作者、分类、版本等）。
3. 执行导出，生成标准化 `.bundle`。
4. 在码镜客户端导入并验证播放效果。

## 与码镜配合

工坊与客户端通过统一壁纸包协议协同：

- 工坊负责内容生产与打包；
- 码镜负责加载、播放与多屏配置；
- 协议变更时建议优先更新工坊导出模板与客户端兼容映射。

## 常见问题

- 导出成功但客户端无法识别：优先检查 `info.plist` 与目录结构是否符合协议。
- 预览正常但实际播放异常：检查资源路径是否为 bundle 根目录相对路径。
- 多屏显示不一致：确认场景 `scaleMode` 与各屏幕分辨率策略。
