## 页面内容

### overview

#### 标题
产品概述

#### 内容
码镜是面向 macOS 的动态壁纸客户端，支持视频与 Scene2D 壁纸包格式。产品强调本地优先和轻量化管理，便于用户在多屏环境下快速切换、预览与应用动态桌面内容。

### install

#### 标题
安装与更新

#### 内容
建议通过 Mac App Store 安装码镜，获取稳定版本与自动更新能力。若后续提供独立发行包，可在此补充系统版本要求、安装步骤、权限说明与升级策略。

### hub

#### 标题
壁纸中心

#### 内容
用户可在壁纸中心浏览官方推荐与精选内容，并将壁纸包导入本地使用。建议在该模块补充筛选、预览、导入、删除与异常处理流程，帮助用户快速完成内容管理。

### playback

#### 标题
播放与控制

#### 内容
码镜支持常见播放控制能力，包括暂停、静音、音量调节与屏幕级应用。建议在此说明“不同屏幕独立配置”的行为边界，以及锁屏、全屏应用、电池模式等场景下的自动策略。

### faq

#### 标题
常见问题

#### 内容
可在此集中维护安装失败、壁纸包导入失败、播放卡顿、无声音、黑屏等问题的排查步骤，确保用户无需跳转多个页面即可完成自助诊断。
